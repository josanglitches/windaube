# 💩 Windaube — Édition Intégrale 2026

French cybersecurity parodic OS sandbox game, educational and fun! Portable app packaged with Node.js/Electron
Find the vulnerabilities, protect against threats, unblock and play mini-games... and more!

Simulateur parodique d'OS et jeu éducatif de cybersécurité.

## 🎮 C'est quoi ?

Windaube est un faux système d'exploitation qui vous apprend la cybersécurité 
en jouant. Trouvez les vulnérabilités, protégez-vous des menaces, débloquez et jouez à des mini-jeux... et plus encore!

## 🛡️ Fonctionnalités

- Bureau interactif façon Windows avec apps, fichiers et menus
- Scénarios de phishing, ransomware, brute force...
- Système de score, missions et progression
- Quiz de cybersécurité en fin de partie
- Aucune donnée collectée (requiert quand même un accès internet car le jeu utilise Google fonts)

## 🛠️ Stack technique

- HTML / CSS / JavaScript (single-file, ~31 000 lignes)
- Packagé avec Electron + electron-builder
- Web Audio API pour les effets sonores
